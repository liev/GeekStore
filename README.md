# GeekStore
eekStore: Marketplace P2P para coleccionistas de TCG, videojuegos y cómics. Permite compra, venta e intercambio con importación desde Moxfield y pagos vía PayPal. Incluye moderación de imágenes por IA (Gemini Vision), panel administrativo robusto y sistema de disputas integrado. Backend en .NET Core y Frontend en React + TypeScript
